Academic Projects

A comprehensive overview of my research projects across astrophysics, quantum optics, and engineering.

Gravitational Wave Astronomy - AMPLFI Project (MIT)

As a researcher at MIT, I worked on the AMPLFI project focusing on gravitational wave detection. My contributions included:

  • Benchmarking multiple waveform generation models such as IMRPhenomD, optimizing detection efficiency by reducing overfitting and computation waste by 40%.
  • Designing a data resampling strategy, improving data processing throughput by threefold in large-scale datasets.
  • Enhancing machine learning models for gravitational wave merger event fitting, achieving an 80% improvement in detection accuracy.
  • Contributing to real-time parameter estimation for gravitational wave sources.

This project emphasized cutting-edge computational astrophysics, focusing on robust data analysis and AI-assisted methods.

Greenhouse Monitoring and Control System - Capstone Project

As part of my undergraduate capstone at Northern Arizona University, I led the design and development of an automated greenhouse monitoring system. Highlights include:

  • Developed hardware modules to monitor environmental factors like temperature, humidity, and light.
  • Implemented an automated control system using microcontrollers to regulate ventilation, lighting, and irrigation.
  • Conducted extensive testing under various conditions, ensuring system reliability and scalability.
  • Managed team coordination and task delegation, ensuring project completion within deadlines.

This experience enhanced my skills in embedded systems, IoT applications, and project management.

Quantum Key Distribution Systems - QuantumCTek

At QuantumCTek, I contributed to the development of next-generation quantum encryption systems. My responsibilities included:

  • Developing photon-encoded quantum communication protocols for enhanced security in data transmission.
  • Optimizing quantum key distribution (QKD) systems to ensure real-world scalability and reliability.
  • Participating in cross-functional teams to integrate QKD technologies into practical networking environments.

This role underscored my expertise in quantum optics and secure communication technologies.
